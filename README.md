# 🧪 Astana Hub Education: Задание №1
## Тестирование смарт-контракта на Rust (Solana/Anchor)

Данный репозиторий содержит решение первого задания курса по разработке на Solana. Проект демонстрирует навыки написания Unit-тестов для проверки базовой логики токен-программы.

### 📋 Описание задания
Реализовать и протестировать основные функции управления токенами:
1. Создание аккаунта пользователя.
2. Минтинг (выпуск) токенов.
3. Перевод токенов между аккаунтами с проверкой баланса.

### 🛠 Реализованные тесты
* **`test_create_token_account`**: Успешная инициализация структуры аккаунта.
* **`test_mint_tokens`**: Проверка изменения состояния баланса после минтинга.
* **`test_transfer_tokens`**: Логика перевода (атомарность списания и зачисления).
* **`test_insufficient_funds`** *(дополнительно)*: Обработка ошибок при нехватке средств.
